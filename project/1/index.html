平台

steam

steam平台是全球最大的综合性数字发行平台之一

玩家可以在steam平台购买、下载、讨论、上传和分享游戏和软件

在steam平台,玩家可以享受许许多多限时原价的优惠和多季度的促销

每次促销时,steam总会掏空玩家钱包里的钱

steam的网络好像总是不太好,每次都需要连接不短的时间

steam几乎每天都会更新,提醒玩家更新并重新启动steam

steam总是会在玩家不知道的时候开始自动更新游戏

xbox

Xbox是一款微软开发的应用程序

Windows 10平台版本在Windows 10预览版10114发布

Xbox可以浏览Xbox Live成就,观看Twitch游戏直播,与好友聊天、以及游戏录像(Game DVR)功能

xbox在pc端提供了很多的游戏

xbox推出了xbox game pass 会员服务以便玩家体验所有xbox提供的游戏

xbox game pass 首个月仅仅收费10港币

epic

Epic Games是近十年来最负盛名的游戏制作团队之一

epic总是会在每周推出一到两款可免费购买的游戏

epic的游戏折扣在平台中是最大的

epic的网络环境同样不太好,即使使用加速器也需要等待不少时间

epic的游戏很多都与其他平台不互通,这让免费游玩联机游戏的玩家十分苦恼

epic可以说是玩家最爱的白嫖游戏平台了
